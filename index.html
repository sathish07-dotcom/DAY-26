Function in js

In JavaScript, functions are blocks of code that can be defined and executed when called.
Functions are used to perform specific tasks, make code more reusable, and organize it into logical sections.

Function Declaration
  • A function can be declared using the `function` keyword followed by the function name and parentheses.

    • function greet(name) {
    console.log("Hello, " + name + "!");
    }
    // Calling the function
    greet('Alice'); // Output: Hello, Alice!

    Function Expression
  • You can also create functions by assigning them to variables.

    • const greet = function(name) {
    console.log("Hello, " + name + "!");
    };
    greet('Bob'); // Output: Hello, Bob!

    Arrow Functions
  • ES6 introduced arrow functions, which are a shorter syntax for writing functions.

    • const greet = (name) => {
    console.log("Hello, " + name + "!");
    };
    greet('Charlie'); // Output: Hello, Charlie!

  • For single expressions, you can omit the curly braces `{}` and the `return` statement.

    • const greet = (name) => "Hello, " + name + "!";
    console.log(greet('Dave')); // Output: Hello, Dave!

    Parameters and Arguments
  • Functions can accept parameters and arguments, allowing you to pass values to be used inside the function.

    • function add(a, b) {
    return a + b;
    }
    console.log(add(2, 3)); // Output: 5

    Return Statement
  • A function can return a value using the `return` keyword. If no `return` is specified, the function returns `undefined`.

    • function multiply(a, b) {
    return a * b;
    }
    console.log(multiply(3, 4)); // Output: 12

    Function Scope
  • Variables declared inside a function are local to that function, meaning they are not accessible outside.

    • function testScope() {
    let localVar = 'I am local';
    console.log(localVar);
    }
    testScope(); // Output: I am local
    // console.log(localVar); // Error: localVar is not defined

    Anonymous Functions
  • Anonymous functions are functions without names, often used for short-lived or callback functions.

    • setTimeout(function() {
    console.log("This runs after 2 seconds");
    }, 2000);

    Default Parameters
  • You can assign default values to function parameters, which are used when no argument is passed.

    • function greet(name = "Guest") {
    console.log("Hello, " + name + "!");
    }
    greet(); // Output: Hello, Guest!
    greet('Eve'); // Output: Hello, Eve!

    Rest Parameters
  • Rest parameters allow you to pass an indefinite number of arguments as an array.

    • function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
    }
    console.log(sum(1, 2, 3, 4)); // Output: 10

    Function Hoisting
  • Function declarations are hoisted to the top of their scope, meaning you can call them before they are defined.

    • console.log(greet('George')); // Output: Hello, George!
    function greet(name) {
    return "Hello, " + name;
    }

  • However, function expressions are not hoisted in the same way:

    • console.log(greet('George')); // Error: greet is not a function
    const greet = function(name) {
    return "Hello, " + name;
    };
